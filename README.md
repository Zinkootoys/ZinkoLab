# ZinkooLab GitHub Pages

Static bilingual site for ZinkooLab.

## Before publishing

1. Replace the pending support-email text in `support.html` with the final email address.
2. Optionally replace the "Coming soon" button with the Google Play URL after publication.
3. Review the privacy policy whenever the app adds ads, analytics, accounts, cloud sync, file uploads or third-party SDKs.

## Publish with GitHub Pages

1. Create a public GitHub repository (for example `zinkoolab-site`).
2. Upload all files in this folder to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select branch `main` and folder `/ (root)`, then save.

GitHub will provide a public HTTPS URL.
